# Helligkeit-Farben-berechnen
List of colors coded as hexadecimals. 

## Goal:
script selects the brightest color from a list of color values and output the red, green and blue components individually.
Bonus: Name the brightest color ('https://www.csscolorsapi.com/')




### Example input: 
list = ["#AABBCC", "#154331", "#A0B1C2", "#000000", "#FFFFFF"]

### Output to the example input: 
"The brightest color is: #FFFFFF (r=255, g=255, b=255)".

alternatively: 
"The brightest color is: #FFFFFF (r=255, g=255, b=255), called White".
